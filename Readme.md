# 📚 Library Management System (Java Console Application)

A simple Java-based console application to manage library operations such as adding books, viewing available books, and deleting books using MySQL database.

## 🔹 Features
- Add new books
- View all books
- Delete books
- MySQL database integration
- JDBC connectivity
- Console-based menu-driven system

## 🔹 Technologies Used
- Java
- JDBC
- MySQL
- VS Code
- Git & GitHub

## 🔹 Project Type
Console Based Application
